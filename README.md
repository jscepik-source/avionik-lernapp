# ✈ Avionik Lern-App

Interaktive Klausurvorbereitung (HS RheinMain, Avionik SS26) – **684 Single-Choice-Fragen**
mit den Modi **Üben · Karteikarten · Test · Lernen (Leitner) · Statistik**.
Läuft komplett im Browser, offline, Fortschritt wird lokal gespeichert. Installierbar als App (PWA).

## Live
Nach dem Aktivieren von GitHub Pages erreichbar unter:
`https://<DEIN-USERNAME>.github.io/avionik-lernapp/`

## GitHub Pages aktivieren
1. Dieses Repo auf GitHub anlegen und die Dateien hochladen.
2. **Settings → Pages → Build and deployment → Source: „Deploy from a branch"**.
3. Branch: **main**, Ordner: **/(root)** → **Save**.
4. Nach ~1 Minute ist die App unter obiger URL online.

## Als App installieren
- **iPhone/iPad (Safari):** Teilen → „Zum Home-Bildschirm".
- **Android (Chrome):** Menü (⋮) → „App installieren".

## Dateien
- `index.html` – die komplette App
- `manifest.webmanifest`, `sw.js`, `icon-*.png` – PWA (Icon, Offline)
